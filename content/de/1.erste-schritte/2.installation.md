## Einrichten

Verwende den `create`-Befehl, um ein neues Projekt einzurichten. Es stehen zwei Templates zur Verfügung, je nachdem ob du Multiplayer-Unterstützung benötigst.

### Single-Player

```bash
demnächst verfügbar
```

### Multiplayer

```bash
npx @mavonengine/create-bootstrap
```

Das Multiplayer-Template enthält die autoritativen Server-Setup, Netzwerk-Boilerplate sowie eine fertige Client/Server-Aufteilung.
